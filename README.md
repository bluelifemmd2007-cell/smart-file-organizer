# Smart File Organizer

A Python tool that automatically organizes files in a folder based on their type.

## Features
- Detects file types (image, video, audio, document, archive, program)
- Creates folders automatically
- Moves files to the correct folder
- Fully customizable via `config.json`
- Clean and simple project structure
